VCF QC Minimum Work Example¶

Rename Chromosomes¶

Initially, this step creates a conversion file with pairs in the format: old_name new_name\n

This file is used in bcftools to take the input vcf file (00-All.vcf.gz) and ass the chromosome names (chr#) to the file, and output that resulting file as 00-All.add_chrs.vcf.gz). Tabix is then used to create an index file (00-All.add_chr.vcf.gz.tbi) from the previous output file, this allows for quick data retrieval. Lastly, the conversion file that was made is removed.

Note: LFS system has walltime in the format HH:MM, so I can't accept the SOS format of HH:MM:SS without modification.

In [2]:
module purge;
module load singularity;
conda activate xqtl;
sos run VCF_QC.ipynb rename_chrs --genoFile reference_data/00-All.vcf.gz --cwd reference_data --container container/bioinfo.sif -c config.yml -q lfs &
